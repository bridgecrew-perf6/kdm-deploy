## Deployment

Download zip file at [here](https://github.com/thamardaw/kdm-deploy).

## Lab Deploy

- Change directory into kdm-lab-deploy.

- Pull remote images.

```
$ docker-compose pull
```

- Start services.

```
$ docker-compose up -d
```
- Stop services.

```
$ docker-compose down
```
